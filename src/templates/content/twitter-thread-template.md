# Twitter/X Thread Template

## Format Guidelines

- **Tweet Limit**: 280 characters per tweet
- **Thread Length**: 5-15 tweets (sweet spot: 7-10)
- **Numbering**: Use "1/" format for clarity
- **Last Tweet**: Summary + CTA + retweet of first

---

## Structure

### Tweet 1: Hook
[Attention-grabbing opening that promises value]

[What they'll learn or why they should read]

1/

---

### Tweet 2-N: Content
[Each tweet = one complete idea]
[Use numbers, bullets, or arrows for lists]

N/

---

### Final Tweet: Summary + CTA
[Recap the value]
[Clear call to action]
[Invite retweet or follow]

## Tips

- First tweet = everything (hooks determine if people read)
- Each tweet should stand alone
- Use line breaks within tweets
- Include visuals/screenshots when relevant
- End with clear CTA (follow, retweet, bookmark)
- Engage with replies for algorithm boost
