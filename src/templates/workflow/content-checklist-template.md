## Content Quality

### Hook & Opening
- [ ] First line grabs attention
- [ ] Creates curiosity or states clear value
- [ ] Relevant to target audience

### Message Clarity
- [ ] Key message is clear within first 2-3 sentences
- [ ] Single focused topic (not trying to cover too much)
- [ ] Reader knows what they'll learn/gain

### Value Delivery
- [ ] Provides actionable insights
- [ ] Includes specific examples or evidence
- [ ] Reader can apply something immediately

### Structure & Flow
- [ ] Logical progression of ideas
- [ ] Clear transitions between sections
- [ ] Easy to scan/skim

### Call to Action
- [ ] CTA is specific and clear
- [ ] Asks for one thing (not multiple)
- [ ] Easy to act on

---

## Platform Requirements

### LinkedIn
- [ ] Under 3,000 characters (ideal: 1,300-2,000)
- [ ] First line is compelling (shows before "see more")
- [ ] 3-5 relevant hashtags
- [ ] Line breaks for readability

### Twitter/X Thread
- [ ] Each tweet under 280 characters
- [ ] Thread numbered clearly
- [ ] First tweet hooks, last tweet has CTA
- [ ] 5-15 tweets total

### Telegram
- [ ] Appropriate length for channel style
- [ ] Formatting (bold, italic) used effectively
- [ ] Engagement prompt included

---

## Constitution Alignment

### Voice & Tone
- [ ] Matches defined voice characteristics
- [ ] Consistent throughout
- [ ] Appropriate for topic and audience

### Content Principles
- [ ] Follows stated content principles
- [ ] Aligns with brand values
- [ ] Authentic to author's expertise

### Audience Fit
- [ ] Appropriate complexity level
- [ ] Addresses audience pain points
- [ ] Uses language they understand

---

## Technical Quality

### Grammar & Spelling
- [ ] No spelling errors
- [ ] Proper grammar
- [ ] Consistent punctuation style

### Formatting
- [ ] Clean line breaks
- [ ] Proper use of bullets/numbers
- [ ] No formatting artifacts

### Links & References
- [ ] All links work (if any)
- [ ] Citations are accurate
- [ ] Mentions/tags are correct

---

## Final Verification

### Quality Bar
- [ ] Would you be proud to publish this?
- [ ] Does it represent your best work?
- [ ] Would you share this if someone else wrote it?

### Pre-publish
- [ ] Read aloud one final time
- [ ] Checked on mobile view (if applicable)
- [ ] Backup copy saved
